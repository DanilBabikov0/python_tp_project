1. Получить палитры пользователя text GET /api/palettes/{username}/?limit=20 Параметры: * limit (int, default: 20, max: 100) - количество палитр Ответ: json [ { "id": "pal1", "name": "Морская тема", "colors": ["#1E90FF", "#00BFFF", "#87CEEB"] }, { "id": "pal2", "name": "Осенняя палитра", "colors": ["#8B4513", "#D2691E", "#FF7F50"] } ] 2. Сохранить палитру text POST /api/palettes/{username}/ Content-Type: application/json Тело запроса: json { "name": "Новая палитра", "colors": ["#FF5733", "#33FF57", "#3357FF"] } Ответ: json { "id": "pal3", "name": "Новая палитра", "colors": ["#FF5733", "#33FF57", "#3357FF"] } 3. Получить конкретную палитру text GET /api/palettes/{username}/{palette_id}/ Ответ: json { "id": "pal1", "name": "Морская тема", "colors": ["#1E90FF", "#00BFFF", "#87CEEB"] } 4. Удалить палитру text DELETE /api/palettes/{username}/{palette_id}/ Ответ: json { "success": true } 5. Случайная палитра text GET /api/palettes/random/?count=5 Параметры: * count (int, default: 5, min: 2, max: 10) - количество цветов в палитре Ответ: json { "colors": ["#FF5733", "#33FF57", "#3357FF", "#F333FF", "#57FF33"] } 6. Сгенерировать палитру text GET /api/palettes/generate/?base=FF5733&count=5&type=complementary Параметры: * base (string, required) - базовый цвет в HEX (с # или без) * count (int, default: 5) - количество цветов * type (string, default: analogous) - тип палитры: * analogous - аналогичные цвета * complementary - комплементарные * triadic - триадные * monochromatic - монохромные Ответ: json { "colors": ["#FF5733", "#33FF57", "#FF33F3", "#5733FF", "#33FFF3"] } 7. Извлечь цвета из изображения text POST /api/palettes/extract/ Content-Type: multipart/form-data Параметры формы: * image (file, required) - файл изображения * count (int, default: 5) - количество извлекаемых цветов Ответ: json { "colors": ["#2F4F4F", "#8B4513", "#D2691E", "#F4A460", "#708090"] }
